# Quad Backend

This directory contains the Quad backend API + realtime server.

## Documentation

Backend-specific documentation lives in:

- `backend/docs/README.md`

Repo-wide (shared) documentation lives in:

- `docs/README.md`

Frontend-specific documentation lives in:

- `frontend/docs/README.md`

## Quick start

From `backend/`:

```bash
npm install
npm run dev
```

Useful endpoints:

- `GET /health`
- `GET /api-docs`
- API routes are under `/api/*`
